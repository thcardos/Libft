# <code style="color : purple">Libft</code>

## <code style="color : purple">Description</code>

Libft es una biblioteca de funciones en C que recrea las funciones estándar de la biblioteca de C, además de incluir funciones adicionales útiles para futuros proyectos de 42. Este proyecto constituye el primer proyecto del currículo de 42 y tiene como objetivo familiarizarse con la programación en C, la gestión de memoria y las estructuras de datos básicas.

El objetivo principal es comprender cómo funcionan internamente las funciones estándar de C y desarrollar una base sólida de conocimientos sobre manipulación de cadenas, memoria y listas enlazadas.

## <code style="color : purple">Instructions</code>

### <code style="color : purple">Available commands</code>

- `make` - Compila la librería
- `make clean` - Elimina los archivos objeto (.o)
- `make fclean` - Elimina los archivos objeto y la librería
- `make re` - Re-compila todo desde cero
- `make bonus` - Compila las funciones bonus (listas enlazadas)

### <code style="color : purple">Uso de IA</code>

Durante el desarrollo de este proyecto, **no se ha utilizado inteligencia artificial** para la generación de código. Siguiendo las directrices de 42, el aprendizaje de los fundamentos requiere un esfuerzo intelectual real y la construcción de conocimientos sólidos sin atajos.

El enfoque adoptado ha sido:
- Lectura de la documentación oficial y manuales de C (`man` command)
- Aprendizaje entre pares mediante discusiones con otros estudiantes
- Experimentación práctica y resolución de problemas de forma autónoma
- Consulta de recursos educativos tradicionales (libros, documentación, tutoriales)
- Uso de herramientas de debugging (gdb, valgrind) para comprender el funcionamiento

Este método permite desarrollar habilidades de razonamiento lógico, comprensión profunda de los conceptos y capacidad para resolver problemas de forma independiente, competencias esenciales para el resto del currículo de 42 y para afrontar los exámenes sin dependencias externas.
